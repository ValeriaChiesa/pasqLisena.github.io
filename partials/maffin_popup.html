MaFFiN - Media Fragment Filesystem Node.JS Server

a project by
  • P. Lisena
2014
  • GitHub Repo
  • Presentation @PoliTO

MaFFiN is a Node.JS server that supports the new following the Media Fragment W3C Reccomandation.
It supports temporal (npt) and track fragment, both in hash and query format.
It works thanks to the video processing tool ffmpeg and the nosql database MongoDB.
